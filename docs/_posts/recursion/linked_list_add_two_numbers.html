* Example of Pre Recursion --- ---
Problem: Given two numbers using Linked List in reverse order. Return their sum as a Linked List
Example: Suppose two numbers are 567 and 489. In Linked list they are represented in reverse.
Number1 = 7 -> 6 -> 5 Number2 = 9 -> 8 -> 4
Desired Output: 6 -> 5 -> 0 -> 1

Recursion Visual:
Recursion
{% highlight c++ %} ListNode *pre_recursion(ListNode *l1, ListNode *l2, int carry) { if(l1 == NULL && l2 == NULL && carry == 0) return NULL; int sum = (l1 == NULL?0:l1->val) + (l2 == NULL?0:l2->val) + carry; carry = sum / 10; sum %= 10; ListNode *head = new ListNode(sum); head->next = pre_recursion(l1 == NULL?l1:l1->next, l2 == NULL?l2:l2->next, carry); return head; } {% endhighlight %}
