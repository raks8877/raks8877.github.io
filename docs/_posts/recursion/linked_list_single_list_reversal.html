* Example of Post Recursion --- ---
Problem: Given a Linked List, return the reversed list.
Example: Suppose the given Linked List is: 1 -> 2 -> 3 -> 4 -> 5
Desired Output: 5 -> 4 -> 3 -> 2 -> 1

Recursion Visual:
Recursion
{% highlight c++ %} ListNode* post_recursion(ListNode* head) { if (head == NULL || head->next == NULL) return head; ListNode *p = post_recursion(head->next); head->next->next = head; head->next = NULL; return p; } {% endhighlight %}
